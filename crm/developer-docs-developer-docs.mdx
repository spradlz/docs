**In the app:** CRM sidebar → **Developer** section → **Developer Docs**.

## What it does

"Developer Docs" is the last item in the CRM's left sidebar. It is a plain external link, not an in-app page: clicking it opens `https://houseloop.mintlify.app` (this documentation site) in a new browser tab. It gives anyone inside the CRM a one-click path from the app to the developer docs.

Because it is a hyperlink, it has no in-app screen of its own — there is no CRM route, page component, form, API call, or database read behind it. It renders as an anchor (`<a href="https://houseloop.mintlify.app" target="_blank" rel="noopener noreferrer">`) with the `Code2` icon, in its own `nav.hl-sidebar-section` labelled "Developer".

## What you configure

Nothing is configurable at runtime — there are no settings, toggles, or inputs. The only thing a developer can change is the link itself, in code:

| What | Where | Notes |
|------|-------|-------|
| Destination URL | `app/components/Sidebar/Sidebar.tsx` (the `<a href="…">` in the Developer nav section) | Hard-coded to `https://houseloop.mintlify.app`. |
| Label / icon | Same anchor in `Sidebar.tsx` | Text "Developer Docs"; icon `Code2` from `lucide-react`. Label is hidden when the sidebar is collapsed (shown as a `title` tooltip instead). |

## Data source

None. This is a static external link with no backend:

- **Routes/APIs:** none. No route exists under `app/(crm)/` for developer docs; the link points to an external host.
- **Database tables:** none. No `hl_*` table is read or written.

The documentation content itself lives in this repo under `docs-site/` and is published to `houseloop.mintlify.app` by the docs mirror — that is a separate build/deploy pipeline, not something the CRM app serves.

## How it fits

This item is the bridge from the running CRM back to its own documentation. It sits in the CRM sidebar (`app/components/Sidebar/Sidebar.tsx`) directly below the Run Log and AI Cost items, in a dedicated "Developer" section, so developers working in the app can jump straight to the docs.

<Note>
The docs nav generator (`tools/docs/crm-nav-sync.mjs`) originally mapped this sidebar item to `app/(crm)/dashboard/page.tsx` as an auto-generated fallback. That mapping is incorrect: `/dashboard` renders the Sisu dashboard (`SisuDashboard`), and "Developer Docs" is not a CRM page at all — it is the external link described above.
</Note>
