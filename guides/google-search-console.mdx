## The diagnostic tool
`tools/seo/gsc-report.mjs` (read-only — uses `GSC_REFRESH_TOKEN`). Pulls Google's OWN data: sitemaps GSC has
processed (+ counts/errors), per-page URL Inspection (`verdict`, `coverageState`, `lastCrawlTime`, Google's
canonical), and search-analytics clicks/impressions.

```bash
node tools/seo/gsc-report.mjs
```

## The submit script — `tools/seo/gsc-submit-sitemap.mjs`
Mints an SA access token (the SA acts **as itself** — JWT `iss` = SA email, **no `sub`**, scope
`https://www.googleapis.com/auth/webmasters`, RS256-signed), then `PUT`s the sitemap to
`webmasters/v3/sites/sc-domain:houseloop.ai/sitemaps/<encoded url>`. **Returned 204 on 2026-06-27** — sitemap
registered, `lastDownloaded=2026-06-27`, 0 warnings / 0 errors, Google processing. `--status` reads without
submitting. (Earlier 403 → fixed by adding the SA as a GSC Owner.) Reuse this same SA-as-itself auth pattern for
any GSC write.

## The weekly monitor — `seoIndexMonitor` (Inngest)
Registered Inngest fn `seo-index-monitor` in `lib/inngest/functions/seo-index-monitor.ts` (cron `0 14 * * 1` =
Mon 14:00 UTC ≈ 8am CT, plus manual event `seo-index-monitor/manual.run`). Each run re-submits the sitemap, pulls
GSC's index data (sitemap child counts + a sample live `/home` page's crawl/index verdict + 28-day traffic), and
Telegram-pings Zach the climb. Auto-captured in the CRM Run Log (capture-fn subscribes to `function.finished`).
Auth via `mintSAAccessToken()` (SA-as-itself) in `lib/google/dwd.ts`.

**Verify-run 2026-06-27** — dispatched `seo-index-monitor/manual.run` → ran in Inngest cloud, run_id
`01KW517NWQS4ATAE9G0G8MVDVZ`, **status=success**, in the Run Log. Payload: `submit ok 204`, sitemap errors 0,
sample page baseline `coverage="URL is unknown to Google" · lastCrawl=NEVER` (the expected pre-crawl baseline),
traffic 31 clicks / 2,259 impressions.

## Status (2026-06-27) — the #1 indexing blocker is FIXED
**The finding (measured from GSC's own API):** "Google can't index what it can't discover." The homepage was
indexed and crawled daily — but **the sitemap was never submitted to GSC**, so Google had **never crawled a
single `/home/<slug>` property page** ("URL is unknown to Google, last crawl = NEVER"). The ~234k property pages
(11 child sitemaps × ~21k) contributed ~nothing. 28-day traffic ≈ 31 clicks / 2,259 impressions, essentially all
homepage.
