## The diagnostic tool
`tools/seo/gsc-report.mjs` (read-only — uses `GSC_REFRESH_TOKEN`). Pulls Google's OWN data: sitemaps GSC has
processed (+ counts/errors), per-page URL Inspection (`verdict`, `coverageState`, `lastCrawlTime`, Google's
canonical), and search-analytics clicks/impressions.

```bash
node tools/seo/gsc-report.mjs
```

The SA **write** path (mint a `webmasters` token from the SA key → `PUT
.../sites/sc-domain:houseloop.ai/sitemaps/<encoded url>`) returns **204** now that the SA is an owner — reuse that
auth pattern for the weekly automation.

## Status (2026-06-27) — the #1 indexing blocker is FIXED
**Before:** Google had crawled **zero** of our ~234k property pages because **the sitemap was never submitted to
GSC** (a property page inspected as "URL is unknown to Google, last crawl = NEVER"; the homepage was the only
indexed page; ~31 clicks / ~2,259 impressions / 28d, essentially all homepage).
