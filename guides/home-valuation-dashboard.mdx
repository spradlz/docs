The Home Valuation Dashboard shows every past client with a HouseLoop property profile. This is the command center for monthly home value reports.

## Coming Soon

This page will contain:

- **Past clients table** — all 1,700+ past clients from SISU import, sortable by name, address, last report date, value change
- **Property profile status** — which clients have photos scraped, CMA computed, profile card built
- **Report history** — when each client last received a home value report, open/click rates
- **Bulk actions** — send reports, refresh CMAs, re-scrape photos

## Data Sources

| Data | Source | Status |
|------|--------|--------|
| Past client list | SISU import (`hl_transactions`, 11,636 rows) | Live |
| Property photos | Zillow/Realtor.com scraper → R2 | Building |
| CMA valuations | Repliers sold comps + geo market stats | Available |
| Purchase price/date | SISU transaction data | Live |
| Market stats | `hl_geo_market_stats` (county/city/zip/neighborhood) | Live |
