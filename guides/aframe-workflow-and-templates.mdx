## The automation spine

A-Frame's automation = **Task templates (the checklist) + Email templates (the comms) + Attachment templates (the doc checklist) + Date templates (the deadline set)**, all folder-grouped and split Buyer (B) / Seller (S) / Listing. Tasks are the trigger; emails attach to tasks as actions; documents and dates track completion. Loading: each builder POSTs to `<Type>TemplatesWidget_show.action`; each item body comes from `<Type>TemplateWidget_show.action?templateId=`.

## Task pipelines (the workflow) — 23 task folders

Every transaction runs a phased checklist. Buyer and Seller each get an identical **8-stage** pipeline; listings get a **5-stage** pre-market pipeline.

| # | Buyer pipeline | Seller pipeline |
|---|---|---|
| 1 | Intake | Intake |
| 2 | Contract Review | Contract Review |
| 3 | Scheduling | Scheduling |
| 4 | Communication | Communication |
| 5 | Compliance | Compliance |
| 6 | Inspections | Inspections |
| 7 | Appraisal | Appraisal |
| 8 | Closing | Closing |
| — | Misc | Misc |

**Listing pipeline:** Stage 1 (within 24h of ERTS signed) → Stage 2 (7-10 days before going live) → Stage 3 (3-4 days before live) → Stage 4 (listing goes live) → Stage 5 (weekend after going live).

## Date templates — 2

```
[8125] Contract Dates Template
[8288] Listing Dates Template
```

Each date template seeds a set of milestone **events** onto the transaction (Under Contract, EMD Due, Appraisal, inspection-resolution, Financing, Final Walkthrough, Close, Possession, Expire) — each completable or omittable, with a per-role visibility flag.

## Email templates — 84 (full bodies in the [Email Library](/guides/aframe-email-library))

Categories: New Contract intros (buyer/seller/title/lender/other-side), EMD/title/HOA/home-warranty requests, inspection scheduling, the appraisal chain, closing (details/time/ALTA/funding/utilities), after-closing + referral-agent + review requests, listing/open-house reminders, and 4 SYSTEM templates (CDA Letter, Listing Summary, Published Activities, Closing Summary).
