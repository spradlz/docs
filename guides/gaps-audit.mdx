## OpenAPI Coverage â€” CLOSED âś…

As of 2026-05-18 all 82 live endpoints are specified in
`apps/houseloop-crm/docs/openapi.yaml`. The previous 49-endpoint gap (BB,
calendar auth, email infrastructure, phone, auth helpers, notifications,

```bash

# documented paths
grep -cE "^  /[a-z]" apps/houseloop-crm/docs/openapi.yaml
```

These should match (modulo the live-endpoint counter including duplicate
methods on a single path). If you add a new `route.ts`, add the spec to
`openapi.yaml` in the same commit â€” the pre-commit hook does NOT enforce
this yet, so do it manually.
