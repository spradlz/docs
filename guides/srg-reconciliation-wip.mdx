## The model (locked 2026-06-27)

**Make SISU complete.** SISU is canon for *which deals exist*, but it's full of blanks; each source fills one kind:

- **SISU** — the deal spine (client, $, close date). A deal exists or it doesn't.
- **MLS** (`hl_srg_mls_deals`, joins 100% by `mls_number`) — the **true address / city / zip** (SISU's address is unreliable, always cross-reference the MLS closing).
- **FUB** — the notes / investigative layer (current home, referral, moved out of state).
- **dotloop** — the signed contract → the real agent (recovers departed agents) when an agent is in question.
- **County deed** — disposition truth: who's on title now. **The router.**
- **Google Places** — geo normalizer when MLS lacks a zip.

## The confidence-tier cost play

Instead of paying the ~$0.50 LLM resolver on all 1,834, classify each client and resolve the cheapest way:

| Tier | Definition | Engine | Cost |
|---|---|---|---|
| 🟢 **GREEN** | 1–2 deals · real named agent · triangulated MLS home | `deterministic-resolve.mjs` (pure assembly) | **$0** |
| 🟡 **YELLOW** | 1–2 deals · agent is Ana-only (departed agent hidden) | assembly + one `dotloop_contract` probe | cheap |
| 🔴 **RED** | 3+ deals, ambiguous agents, or a deal missing triangulation | `s6-investigator.mjs` (Sonnet) | ~$0.30–0.50 |
| 🚫 **DNC** | surname on the SRG agent roster + COI signal | DNC card | n/a |

**Within GREEN, the deed is the router** — and the split is FACT, not a guess:

- **`owner_match`** (client on title) → **lock free, high confidence** — current home = the bought address, deed-confirmed.
- **`owner_differs`** (someone else on title) → **MOVED** — never assert the old address; route to the mover-finder.
- **`ambiguous`** (trust/LLC) → likely still theirs, med, lockable.
- **`not_found` / `out_of_coverage`** → can't verify → coverage pass (below).

## The mover-finder — free, no paid skip-trace

For every `owner_differs` (moved) client, find the new home with free tools. **The county deed is always the verifier.**

1. **`gis_name_search(surname, county)`** — returns every parcel that surname *owns* in-county. Find **and** verify in one shot (deed-level). The strongest signal; expand across the 5 covered counties.
2. **FUB notes/addresses** + **truepeoplesearch** (real-Chrome) — propose a candidate new address.
3. **Verify every candidate** through `verifyOwner()` → only an `owner_match` promotes it to the confirmed current home; otherwise `current_residence = sold_moved_UNVERIFIED` (never a lie).
4. **Outreach backstop** (downstream CRM lane) — the humble reactivation message surfaces the new address from the client directly.

## Doctrine locked (principles, not rules)

1. **SISU's address is unreliable — cross-reference the MLS closing** for every transaction's address/geo.
2. **Never assert a current home the deed contradicts** (the Nations error). Sold/moved ≠ knowing where they live now.
3. **No SISU = not a deal** — never invent a transaction or lost listing from a Repliers or dotloop-only signal.
4. **Chain-of-custody agent = the agent on OUR client's side** (buyer client → buying agent; seller client → listing agent).
5. **Card format:** `CONTACTS → TRANSACTIONS WITH US → CURRENT (deed-confirmed) → DOSSIER`.
