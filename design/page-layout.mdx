## The frame

Every CRM page is built from the same frame, in `app/components/PageShell/PageShell.tsx`:

| Component | It is |
|---|---|
| `PageShell` | The outer surface card + the page contract. |
| `PageHeader` | The **navy** strip at the top — the page title and page-level actions. This is the **only** navy in the app. |
| `PageContent` | The white content zone below the header, where the blocks stack. |

Header actions use `.hl-btn--on-navy` (a translucent-white button that reads on the navy) or `.hl-btn--primary`.

## The rule: a page is a flat stack of blocks

Below the navy header, a page is a **flat, top-to-bottom stack** of blocks. Each block is one of the five body types, and **each carries its own brand-blue header with a description of what it is**. Nothing floats between blocks; nothing nests inside another block.

```
┌─────────────────────────────────────────┐
│  NAVY PageHeader — page title + actions  │   ← the only navy
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  ▌ Brand-blue header — "Automation"      │   ← a Section (rows)
│    Turn Myli's auto follow-up on/off     │
│  ─ AI Follow-Up Engine ............ [ ]  │
│  ─ AI Voice ....................... [x]  │
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  ▌ Brand-blue header — "AI Performance"  │   ← a StatStrip (KPIs)
│    What Myli did — last 30 days          │
│   [ 8 ] [ 18 ] [ 991 ] [ 0 ] [ 1 ]       │
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  ▌ Brand-blue header — "Activity"  (128) │   ← a MenuTable (table + menu)
│    [All AI] [Myli] [Zach]   [filters]    │
│  ─────────────── table ───────────────   │
└─────────────────────────────────────────┘
```

## The color law for layout

- **Page top = navy.** The `PageHeader` strip, and nothing else.
- **Every block header = brand blue** (`--hl-blue`), one step of contrast off the navy.
- There is exactly **one** level of blue below the navy — no lighter-blue sub-bands.
- **Nothing floats.** A control (toggle, selector, filter, KPI) is never a bare card or pill on the page background. Toggles live in a Section; selectors and filters live in a MenuTable header; KPIs live in a StatStrip.
