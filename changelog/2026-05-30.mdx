## Email Template System

Built the AI-assisted email template system for HouseLoop CRM:

- **Full-page template editor** at `/settings/email-templates/new` and `/[id]` with TipTap WYSIWYG, image/link support, and categorized merge field picker (Contact, Listing, Agent, Market, Search)
- **EmailPreviewModal** with Preview/HTML/Desktop/Mobile tabs — renders full production HTML in an iframe exactly as it would appear in a real inbox
- **AI-assisted workflow**: describe the email in plain language, Lee generates the full production HTML (logo, listing cards, CTA, unsubscribe footer), saves to DB, iterate via preview until locked in
- **Template picker** ("Use Template" button) in the campaign composer — select a template to pre-fill subject + body
- Two initial templates: **Listing Alert - Welcome** and **Listing Alert - New Matches**

## Combined Saved Search Alert Email

The saved search system now sends **one combined email** instead of two separate ones:

- Previously: a "Your search is saved" confirmation email + a separate listing alert email fired on search creation
- Now: one email with the confirmation message + matching listing cards + real MLS photos
- The alert engine (`alert/logic.ts`) runs the search through the website's own engine (`parseSearchParams` + `searchFromDb`), honoring all filters including AI vision features (pool, hardwood, etc.)
- Status pills bumped to 13px for better visibility

## HouseLoop Logo SSOT

Locked the HouseLoop logo as a single source of truth:

- One PNG file (`houseloop-logo.png`) in both `houseloop-web/public/` and `houseloop-crm/public/`
- All emails reference `/houseloop-logo.png` (relative URL)
- Banned: inline SVG recreations, emoji substitutes, multiple logo styles
