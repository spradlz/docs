## What it is

Every email HouseLoop sends is a real named row in `hl_templates` (kind `email`) — a **complete
self-contained HTML document** (migration 0300: chrome baked in, so "preview = send," no code assembly). Bodies
build on one of three shells:

- `notify_base` — agent-alert chrome, with the contact card.
- `notify_base_system` — same, no contact card.
- `marketing_base` — the branded 560px card (listing alerts, saved-search, seller).

## How it renders

`buildMergeScope(lead)` → `applyMerge(body, scope)` (`lib/email/template-merge.ts`) resolves `{{namespace.field}}`
tokens (see [Merge Fields](/platform/merge-fields)) + `{{ai.*}}` blocks (see [AI Context Blocks](/platform/ai-context-blocks)),
then sends via Resend/Gmail. Preview endpoint: `resolveForPreview()`; send: `/api/email/send-template`.

## The page: USED IN + SENT (Zach LOCKED 2026-07-02)

`Settings → Email → Email Templates` — a flat group-pill row (**All · Buyer · Seller · Transaction Coordination
· Agent Notifications**, derived by `groupForTemplate`) + the table `TEMPLATE · USED IN · SENT · CREATED · VIEW`:

- **USED IN** (`/api/template-usage` → `resolveTemplateUsage`): the workflow(s) that fire it (active dot), the
  agent notification that sends it, or **"Not used yet"** — the only safe-to-delete signal.
- **SENT**: actual sends from `hl_email_send_log` (click-through to every lead who received it).
