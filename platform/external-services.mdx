<Warning>**Credential law:** every credential lives in `.env` (mirror of the Master Login Credentials
Sheet). The sheet is the SSOT; `sync-creds.mjs` reads it (now A1:G500 — it used to truncate at row 200 and
silently drop keys). Keys with non-derivable env names (DB URLs, tokens) are **aliased** in the sync so they
survive. Never hand-edit a password into `.env` — put it in the Sheet.</Warning>

## Single points of failure

- **Camp David is a SPOF.** The master `fub.*` DB AND its backups live on the same Mac mini's disk — one
  hardware failure loses both. The audit prescribes an off-box (R2) backup sync + retention (exte-3, open).
- **The FUB scrapers** (pm2 fub-backfill-a/b) run only on Camp David; if the box is down, FUB import stalls
  — the cockpit's **FUB Import** probe catches this (red after 24h silent).
