<Note>Part of [Messages & Notifications](/platform/message-catalog). **This is DATA injected INTO a template —
not a template.** One registry: `lib/merge-fields.ts`.</Note>

## What it is

The `{{namespace.field}}` placeholders any template (email or text) can use. **93 tokens across 12 groups** — one
canonical registry (`lib/merge-fields.ts`), so there's no second list to drift. Resolved at send time by
`buildMergeScope(lead)` → `applyMerge(body, scope)` (`lib/email/template-merge.ts`); unknown tokens resolve to
empty (raw braces never leak to a recipient).

## The groups

| Group | Example tokens |
|---|---|
| Contact | `{{contact.first_name}}` · `{{contact.full_name}}` · `{{contact.phone}}` |
| Listing | `{{listing.address}}` · `{{listing.price}}` · `{{listing.beds}}` |
| Agent | `{{agent.full_name}}` · `{{agent.email}}` · `{{agent.signature}}` |
| Market | `{{market.area_name}}` · `{{market.avg_price}}` |
| Search | `{{search.name}}` · `{{search.match_count}}` · `{{search.matches}}` |
| Request | `{{request.address}}` · `{{request.date}}` · `{{request.message}}` |
| Valuation | `{{valuation.estimated_value}}` · `{{valuation.range_low}}`/`range_high` |
| Time & Day | `{{time.signoff}}` · `{{time.day_part}}` (business timezone, at send) |
| Company | `{{company.name}}` · `{{company.address}}` · `{{company.footer}}` (CAN-SPAM) |
| Trigger Event | `{{event.activity}}` — the per-event phrase ("saved a home"), from `hl_trigger_events` |
| Notification | `{{notify.title}}` · `{{notify.body}}` — the computed team-alert headline |
| AI Context | `{{ai.*}}` — see [AI Context Blocks](/platform/ai-context-blocks) (composed, not a static token) |

## Where to browse

Each template editor has a **Merge Field Reference** panel (click a token to copy). The AI Context group is
authored separately at **Settings → AI Context Blocks** (composed per-lead, not a static value).
